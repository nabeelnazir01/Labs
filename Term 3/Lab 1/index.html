Why do we need a Uniform Resource Locator (URL)?

It is used as a reference point and address to a website.

Give an example of Uniform Resource Locator (URL) and specify all components of an URL.

https://www.w3schools.com/html/exercise.asp?filename=exercise_html_attributes1

https = Protocol
www.w3schools.com = Domain
html/exercise.asp = Path
filename=exercise_html_attributes1 = Query String

What is the difference between a static and a dynamic web site?

Static web sites are consited of HTML pages that are the same for all users at all times.
Dynamic web sites are consited of server based programs which can use databases, financial intitution etc and then output to HTML.

Why do we need Domain Name System (DNS)?

It allows users to enter in a website name instead of an ip address.

What is DNS address resolution?

It's a mechanism which discovers the ip address from an DNS.

Give some examples of generic top-Level domains (gTLD).

.com, .net, .net, .info, .biz, .edu

Give some examples of country code top-Level domains (ccTLD).

.ca, .us, .uk, .fr, .jp

What is the function (purpose) of Hypertext Transfer Protocol (HTTP)?

Used to respond to requests with a response code, headers and an optional message which can have files.

Give some examples of HTTP request methods.

GET, HEAD, POST, PUT, DELETE

Define a Web Server.

It's a computer that responds to HTTP requests.
