What are main benefits of using CSS?

The format control is better than HTML, Web sites are more maintainable, More accessible, Faster and can be used for different output mediums.

What are different parts of a CSS style rule?

It consists of a selector and property and value pairs which are called declarations.

What are an element selector and grouped element selector in CSS? Provide an example of each.

An element selector is a way to tell the browser which elements will be affected by the css declarations. Grouped selectors is a way to select multiple elements.

Element:
h2 {
color: red;
}

Group Element:
h2, h3, h4 {
color: yellow;
}

What are class selectors in CSS? What are id selectors in CSS?

Class selectors can target elements using a period (.) and the class name. ID selectors can target elements using a hash (#) and the ID name.

What are contextual selectors in CSS?

Contextual selectors allow you to select elements based on their ancestors, descendants, or siblings

What are the three cascade principles used by browsers when style rules conflict?

CSS uses the following cascade principles to help it deal with conflicts: inheritance, specificity, location.
